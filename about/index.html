                      关于网站

                      飞机群 - Telegram频道群组导航

                      飞机群收录整理了Telegram上的许多优质频道、群组、机器人，旨在打造一个高质量Telegram导航，为用户带来更好的体验。从超过20,000个频道、群组、机器人中精心筛选，Telegram导航，有feijiqun就够了！

                      飞机群分为3个模块： 频道导航 、 群组导航 、 机器人导航 ，分别对应Telegram频道（Channel）、Telegram群组（Group）、Telegram机器人（Robot）。

                      我们非常重视并承诺保护网站访客的隐私，具体请参阅我们的 《隐私政策》 。

                      本项目的构建代码已开源，您可以在 GitHub 上找到网站的源代码，感觉还不错的话Star支持一下吧！

                      由飞机群打造的开源的Telegram知识库，邀请您一起参与维护： TGwiki - Telegram知识库 。

                      支持本项目

                      飞机群的维护需要一定的人力物力，在此过程中离不开大家的支持。
